dumic

Dumic is a POSIX shell utility for routing audio through the input device. The name is a combination of the words "dummy microphone." It works by creating a virtual microphone input with the pactl command used for PulseAudio; after that, you can select any file with audio to route through the virtual device and it will output on the other end.
